# 1.2 机器学习的数学基础

## 学习目标

+   将泛化解释为机器学习的目标。
+   解释复杂和简单模型中的偏差 - 方差权衡。
+   举例说明免费午餐定理。
+   将统计方法解释为第一个 ML 算法。
+   描述 ML 与统计建模的不同之处。
+   使用交叉验证选择超参数。
+   将课程中学到的线性和逻辑回归方法，应用于课程项目中的 FDIC 银行业务呼叫报告数据。
